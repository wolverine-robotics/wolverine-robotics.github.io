### Color Usage Guidelines

> **Pro Tip:** Use **Blue** as your dominant "anchor" color for professional documents and **Orange** as an "accent" for buttons, calls to action, and robot highlights to make them pop.

- **Blue (#0047AB)**: Headers, titles, primary text, professional documents, main backgrounds
- **Orange (#FF8C00)**: Buttons, CTAs, accents, highlights, robot details, energy elements
- **Grey (#A9A9A9)**: Secondary text, borders, subtle backgrounds
- **White (#FFFFFF)**: Page backgrounds, text on dark backgrounds

## 3. Typography

Fonts should be clean, modern, and easily readable from across a crowded pit area.

### Font Stack

| Role | Font Family | Weight | Usage |
|------|-------------|--------|-------|
| **Primary Heading** | Montserrat | Bold (700) | Logo text, title pages, slide headers |
| **Subheaders** | Roboto | Medium (500) | Section titles in Engineering Portfolio |
| **Body Text** | Open Sans | Regular (400) | Long-form descriptions, emails, website copy |

### Typography Styles

#### Heading Style (Montserrat Bold)
- Use **ALL CAPS** for a rugged "Wolverine" feel
- Letter spacing: 0.05em
- Examples: `WOLVERINE ROBOTICS`, `ABOUT US`, `OUR ROBOT`

#### Subheading Style (Roboto Medium)
- Clean and industrial aesthetic
- Use for section titles and navigation
- Examples: "Technical Specifications", "Team Members"

#### Body Text Style (Open Sans Regular)
- Minimum **10pt** for print materials
- Minimum **16px** for web content
- Line height: 1.6 for optimal readability

### Font Resources

Download fonts from [Google Fonts](https://fonts.google.com/):

- [Roboto](https://fonts.google.com/specimen/Roboto)

## 4. Logo Usage & Assets

### Clear Space

Maintain a "safety zone" around the logo equal to the width of the letter **'W'** in your name. This ensures the logo remains prominent and uncluttered.

```
        ┌─────────────────────────────┐
        │         (Clear Zone)        │
        │   ┌───────────────────┐     │
        │   │                   │     │
        │   │  WOLVERINE        │     │
        │   │  ROBOTICS         │     │
        │   │  33791            │     │
        │   │                   │     │
        │   └───────────────────┘     │
        │         (Clear Zone)        │
        └─────────────────────────────┘
```

### Background Guidelines

| Background Color | Logo Version |
|------------------|--------------|
| **Blue backgrounds** | Use white or orange version |
| **Orange backgrounds** | Use blue or black version |
| **White/Light backgrounds** | Use blue primary version |
| **Dark backgrounds** | Use white version |

### Logo Restrictions

❌ **DO NOT:**
- Stretch or distort the logo
- Skew the proportions
- Apply "early 2000s" drop shadows
- Use unapproved color combinations
- Place on busy, cluttered backgrounds
- Make the logo smaller than minimum size (1" wide for print)

✅ **DO:**
- Keep the logo flat and modern
- Maintain original proportions
- Use approved color versions
- Ensure adequate contrast with backgrounds

## 5. Apparel & Pit Design

### Team Shirts

| Element | Specification |
|---------|---------------|
| **Primary Color** | Deep Electric Blue (#0047AB) |
| **Accent Color** | Safety Orange (#FF8C00) for text and details |
| **Front** | Team logo/name + Team number in pocket area |
| **Back** | Team #33791 (prominently displayed) |
| **Optional** | Sponsor logos on sleeves |

### Robot Bumpers

Per FTC rules, your team must have a set of **Red** and **Blue** bumpers.

| Specification | Requirement |
|---------------|-------------|
| **Team Number Font** | High-contrast white |
| **Minimum Height** | 2.5 inches tall |
| **Font Style** | Bold, sans-serif (Montserrat recommended) |
| **Placement** | Centered, clearly visible |

### Pit Banner Design

| Zone | Content | Height |
|------|---------|--------|
| **Eye Level** (5-6 ft) | Team Name + Number | Most prominent |
| **Upper Zone** | School name, Logo | Secondary |
| **Lower Zone** | Sponsor logos | Supporting |

**Key Principles:**
- Most important information at eye level (approximately 5-6 feet high)
- Blue background with white text OR white background with blue text
- Orange accents for visual interest
- High contrast for readability from a distance
